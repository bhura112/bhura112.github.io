Micro Controllers



Microcontroller is a small computer on a single meta-oxide-semiconductor ic (integrated circuit)
or chip. It's also called SOC (small of computer).

These micro computers are used for a specific task. A Microcontroller cantains cpu single core or multi-core ,
memory unit, bootloader, gpio ,Timer, communication interface like : UART, USART, SPI, I2C, Ethernet
and many other features.
